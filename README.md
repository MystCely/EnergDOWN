# EnergDOWN — Caffeine Tracker (School Project)

A multi-view **vanilla JavaScript** project built as part of **GET Academy — Emne 2**.  

---

## Tech Stack

- HTML  
- CSS  
- JavaScript (no frameworks)  
- Model–View–Controller-style structure  
- Local state management  

---

## Purpose

This repository exists as part of a **school team project**, created under time limits and client-driven requirements.  
The goal was to practice:

- DOM manipulation  
- JavaScript state management  
- Multi-page navigation in a single-page structure  
- Working collaboratively on shared features  
- Reading, understanding, and modifying someone else’s code
